## Requirements
### Requirement: Signed Documents Tab

Inbox MUST have a tab showing documents the user has already signed.

#### Scenario: User views signed documents

- GIVEN user navigates to OMK-Docs Inbox
- WHEN user clicks "Sudah Ditandatangani" tab
- THEN list shows all documents user has signed
- AND each entry shows signed date, document title, and requester
