# pdf-tools Spec Delta

## ADDED Requirements

### Requirement: PDF Tools Section
The application SHALL provide a dedicated section for PDF manipulations.

#### Scenario: Sidebar Navigation
- **GIVEN** the user is on the application
- **WHEN** they open the "Tools" menu
- **THEN** a "PDF Editor" section MUST be available
- **AND** it MUST contain links to Compress, Editor, Splitter, and Converter.
