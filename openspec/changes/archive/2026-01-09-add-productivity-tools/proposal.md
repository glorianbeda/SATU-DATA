# Proposal: Add Productivity Tools

## Summary

Menambahkan dua fitur produktivitas baru ke aplikasi Satu Data+:
1. **To-Do List** - Daftar tugas interaktif dengan drag-and-drop untuk menandai selesai/belum
2. **Notebook** - Catatan pribadi dengan rich text editor

## Motivation

Fitur produktivitas ini akan membantu pengguna mengatur pekerjaan mereka langsung di dalam aplikasi tanpa perlu menggunakan tools terpisah.
