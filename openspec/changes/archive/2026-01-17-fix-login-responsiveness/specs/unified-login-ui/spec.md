# unified-login-ui Specification Delta
 
## ADDED Requirements
 
### Requirement: Responsive Layout
 
The authentication forms MUST fit fully within the viewport on mobile devices (down to 320px width) without horizontal scrolling or clipping.

#### Scenario: Mobile View
Given a user accesses the login page on a mobile device (e.g., iPhone SE)
When the page loads
Then the login form is fully visible
And there is no horizontal scrollbar
And the margins adjust to maximize usable space
