# Implement Finance Module

## Summary
Implement a comprehensive Finance Management module including Income, Expense, and Balance management, with dashboard integration and visualization using ApexCharts.

## Motivation
The user requested specific menus for "Manajemen keuangan pemasukan", "Manajemen keuangan pengeluaran", and "Manajemen Saldo saat ini". They also requested using ApexCharts for visualization and the existing DataTable for listings, as well as dashboard integration.

## Impact
-   **Backend**: New `Transaction` model and API endpoints.
-   **Frontend**: New pages for Income, Expense, and Balance; Sidebar updates; Dashboard widgets.
-   **New Feature**: "Quick Edit" page (`/quick-edit`) implementing a spreadsheet-like interface for rapid bulk editing of financial transactions.
